# YouTube Slash Search

A lightweight Brave/Chrome extension that focuses the YouTube search bar when pressing `/`.

This feature already exists on YouTube, but it was inconsistent in my browser, so I decided to build my own version.

I'm also a big fan of keyboard shortcuts, so this project was a fun way to improve my daily browsing experience while learning how browser extensions work.

## Features

- Press `/` to instantly focus the YouTube search bar
- Lightweight and fast
- Works with Brave and Chrome
- Built with vanilla JavaScript

## Installation

1. Clone or download this repository
2. Open `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the extension folder
